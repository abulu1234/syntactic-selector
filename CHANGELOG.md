## [0.2.0] - 2024-XX-XX

### المضافة
- واجهة تفاعلية متقدمة مع خريطة حرارية
- رسم ثلاثي الأبعاد للمقاييس الإحصائية
- مخطط Chord للارتباطات
- شجرة قرار الترجيح
- عرض السياقات النصية التفاعلي
- فلترة ديناميكية حسب الترجيح
- ترتيب متعدد الخيارات
- تصدير JSON للبيانات التفاعلية
- حفظ السياقات النصية الكاملة
- حساب Z-scores للمقاييس
- إحصائيات مفصلة عن الملفات

### التقنية
- Python 3.7+
- pandas لمعالجة البيانات
- openpyxl لتصدير Excel
- JavaScript و D3.js للواجهة التفاعلية
